### 🏠 [Homepage](https://github.com/labstag/core#readme)

## Install dev

```sh
make create-network
make install
make setenv
make install-dev
```

## Install prod

```sh
make create-network
make install
make install-prod
```

## Show stack

```sh
make showstack
```
